# 🏫 การวิเคราะห์และแสดงผลข้อมูลแบบสำรวจอาคารคณะวิทยาศาสตร์

โปรเจกต์ด้าน **Data Visualization** ที่วิเคราะห์ความคิดเห็นของนักศึกษาต่ออาคารเรียน ห้องเรียน และสิ่งอำนวยความสะดวกภายในคณะวิทยาศาสตร์ โดยใช้ภาษา Python เพื่อสำรวจระดับความพึงพอใจ ปัญหาที่พบ และข้อเสนอแนะในการพัฒนาสภาพแวดล้อมสำหรับการเรียนรู้

---

## 📌 ภาพรวมของโปรเจกต์

โปรเจกต์นี้นำข้อมูลจากแบบสอบถามของนักศึกษามาวิเคราะห์ในหัวข้อต่าง ๆ ได้แก่

- ความสะอาดของอาคาร
- สภาพห้องเรียนและห้องน้ำ
- ความปลอดภัยภายในอาคาร
- สิ่งอำนวยความสะดวกสำหรับจักรยานและทางเดินเท้า
- อุปกรณ์ภายในห้องเรียน
- ความพึงพอใจของนักศึกษา
- ห้องเรียนที่ชอบและไม่ชอบ
- ข้อเสนอแนะในการปรับปรุงอาคารและสิ่งอำนวยความสะดวก

วัตถุประสงค์ของโครงการคือการแปลงข้อมูลดิบจากแบบสอบถามให้เป็นข้อมูลเชิงภาพ (Visualization) ที่สามารถนำไปใช้สนับสนุนการตัดสินใจและการพัฒนาสภาพแวดล้อมของคณะได้อย่างมีประสิทธิภาพ

---

## 📊 ชุดข้อมูล (Dataset)

- **แหล่งที่มา:** แบบสอบถามนักศึกษา (Google Forms)
- **จำนวนผู้ตอบแบบสอบถาม:** 102 คน
- **จำนวนตัวแปร (Features):** 31 ตัวแปร

ข้อมูลประกอบด้วย

- ข้อมูลพื้นฐานของผู้ตอบ
- คำถามแบบเลือกตอบ (Multiple Choice)
- ระดับความพึงพอใจ
- ข้อเสนอแนะและความคิดเห็นแบบปลายเปิด

## 📈 การวิเคราะห์ข้อมูล

โปรเจกต์นี้แสดงผลข้อมูลในหลายด้าน ได้แก่

### 👥 ข้อมูลทั่วไปของผู้ตอบแบบสอบถาม

- การกระจายตัวของเพศ
- ชั้นปี
- สาขาวิชา

### 🏢 ปัญหาที่พบในอาคาร

- ความสะอาดของห้องเรียน
- ความสะอาดของห้องน้ำ
- ความสะอาดของอาคาร
- อาคารที่ทำให้รู้สึกไม่ปลอดภัย
- สาเหตุที่ทำให้รู้สึกไม่ปลอดภัย

### 🚲 สภาพแวดล้อมภายในมหาวิทยาลัย

- ปัญหาทางจักรยาน
- ปัญหาทางเดินเท้า

### 🎓 สิ่งอำนวยความสะดวกด้านการเรียน

- ความสว่างภายในอาคาร
- คุณภาพของโปรเจกเตอร์และจอแสดงผล
- สภาพโต๊ะเรียน
- ความพร้อมของคอมพิวเตอร์

### 😊 การวิเคราะห์ความพึงพอใจ

วิเคราะห์ระดับความพึงพอใจของนักศึกษาต่อ

- อาคาร บร.2
- อาคาร บร.3
- อาคาร บร.4
- อาคาร บร.5

### 💬 ความคิดเห็นและข้อเสนอแนะ

วิเคราะห์ข้อมูลเกี่ยวกับ

- ห้องเรียนที่ชอบมากที่สุด
- ห้องเรียนที่ชอบน้อยที่สุด
- เหตุผลที่เลือก
- ข้อเสนอแนะในการปรับปรุง
- อาคารต้นแบบจากมหาวิทยาลัยอื่น

## 📌 ผลลัพธ์ที่ได้จากการวิเคราะห์

การวิเคราะห์ช่วยให้สามารถระบุได้ว่า

- อาคารใดมีระดับความพึงพอใจต่ำที่สุด
- ปัญหาด้านความสะอาดที่ถูกรายงานบ่อยที่สุด
- จุดที่นักศึกษารู้สึกไม่ปลอดภัย
- สิ่งอำนวยความสะดวกที่นักศึกษาต้องการให้ปรับปรุงมากที่สุด
- ภาพรวมของความพึงพอใจต่อสภาพแวดล้อมในการเรียน

ข้อมูลเหล่านี้สามารถนำไปใช้ประกอบการวางแผนพัฒนาอาคารและปรับปรุงสิ่งอำนวยความสะดวกภายในคณะได้

## 📖 สิ่งที่ได้เรียนรู้

จากการทำโปรเจกต์นี้ ได้ฝึกทักษะด้าน

- การทำความสะอาดข้อมูล (Data Cleaning) ด้วย Pandas
- การวิเคราะห์ข้อมูลเชิงสำรวจ (Exploratory Data Analysis: EDA)
- การสร้าง Data Visualization ด้วย Matplotlib และ Seaborn
- การวิเคราะห์ข้อมูลจากแบบสอบถาม (Survey Data Analysis)
- การสื่อสารข้อมูลผ่านการเล่าเรื่องด้วยภาพ (Data Storytelling)
